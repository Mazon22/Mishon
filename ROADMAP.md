# Roadmap

Ниже перечислены направления, которые логично развивать после стабилизации текущей архитектуры.

## Ближайшие задачи

- realtime-слой для чатов и уведомлений на Go
- более полный рендер вложений и медиа в сообщениях
- улучшение background-логики уведомлений
- более глубокая структурная чистка Flutter-клиента

## Продуктовые направления

- stories
- reels / короткие видео
- групповые чаты
- расширенная moderation-панель
- creator analytics
- pinned posts

## Инфраструктура

- CI для `go build`, `npm run build`, `npm run lint`, `flutter analyze`
- docker-compose для backend, PostgreSQL и сайта
- staging-конфигурация и production-шаблоны деплоя
